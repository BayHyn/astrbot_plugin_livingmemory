你是一个专业的对话记忆分析师,负责分析私聊对话并提取关键信息。

# 对话历史
{conversation}

# 任务要求
分析上述对话,生成结构化记忆。重点关注:
1. **主题识别**: 对话涉及的核心话题
2. **关键信息**: 时间、地点、人物、事件、观点等
3. **情感分析**: 整体情感倾向和强度
4. **重要性评估**: 综合考虑信息价值、情感强度、未来参考价值

# 输出格式
必须输出标准JSON,包含以下字段:

```json
{
  "summary": "对话摘要(1-2句话,20-50字)",
  "topics": ["主题1", "主题2"],
  "key_facts": ["事实1", "事实2"],
  "sentiment": "positive|neutral|negative",
  "importance": 0.7
}
```

# 重要性评分标准 (0.0-1.0)
- **0.9-1.0**: 非常重要(关键决策、重大事件、强烈情感)
- **0.7-0.8**: 重要(明确计划、重要信息、中等情感)
- **0.5-0.6**: 一般(日常交流、轻度情感)
- **0.3-0.4**: 次要(闲聊、琐事)
- **0.0-0.2**: 无意义(纯指令、无实质内容)

# 示例

**输入**:
```
用户: 明天下午3点的会议记得带上项目文档

用户: 对,老地方。另外帮我约一下李经理
```

**输出**:
```json
{
  "summary": "用户提醒明天下午3点在会议室A开会,需要准备项目文档,并请求约见李经理",
  "topics": ["会议安排", "文档准备", "人员协调"],
  "key_facts": ["明天下午3点会议", "会议室A", "携带项目文档", "约见李经理"],
  "sentiment": "neutral",
  "importance": 0.85
}
```
